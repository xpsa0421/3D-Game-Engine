# 3D Game Engine
A basic 3D game engine developed with C++ and DirectX.

<br />

### Features
* 3D camera movement
* Basic 3D object rendering
* 3D object transformations
* FBX 3D object rendering
* Diffuse lighting + wireframe
* 3D math
